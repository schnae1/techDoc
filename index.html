Introduction

C++ is a general-purpose language that has many applications. it's object oriented, generic, and functional features are the reasons why so many people use it today.

Often times, people who are learning to program learn C++ first.

Hello World

To get started writing programs in C++ we will first write a simple program that outputs "Hello World!" to the screen.

int main(){
cout << "Hello World!" << endl;
return 0;
}

Along with the code above, the lines "#include < iostream >" and "using namespace std;" must be included for the program to run.

Variables

Variables are used to store values. Each variable has a type and can hold a given value of that type.

The different primitive types variables can hold are:

  • Integer (int)
  • Character (char)
  • Boolean (bool)
  • Floating Point (float)
  • Double Floating Point (double)
  • Void (void)

Some ways to declare and assign values to variables can be seen bellow:

int x = 9;

double y = 2.3;

bool z = false;
